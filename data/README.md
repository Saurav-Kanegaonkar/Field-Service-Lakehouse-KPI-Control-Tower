# Data Sources

Synthetic source-style data for field service lakehouse analytics, KPI integrity, productivity dashboards, discrepancy audits, and regional operations recommendations.

- `entities.csv`: 36 field region records.
- `daily_metrics.csv`: 5,040 field region-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
